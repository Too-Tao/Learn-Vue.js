# 犯了个低级错误
## 自定义事件
### 自定义事件名称不能使用大写，要用 小写 或者 - 隔开
* @getData 错  
  应改成@getdata 或者 @get-data

-----------
1. 子组件传递数据的方式
2. 父组件接收数据的方式

----------------------------------------------------------

1.子组件传递数据的方式
------------------
- 子组件是通过自定义**事件方法**向父组件传递数据
- 子组件中定义: methods: {...} 属性
- 事件方法中,通过this.$emit()方法,将事件与数据发送出来

----------------------------------------------------------

2.父组件接收数据的方式
------------------
- 父组件通过:v-on指令,事件监听
- 在它的事件方法中,获取到子组件的数据
- 最后在实例挂载点模板中输出数据
