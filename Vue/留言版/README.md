## JS 原生
### creteElement(el)
* 创建一个节点，如 createElement(li)，创建一个li标签

* 父标签.appendChild(li)，在父标签下添加一个li子标签
### 父标签.firstElementChild
* 父标签下的第一个儿子标签
### insertBefore(li,first)
* 在一个标签前插入 li 标签
